# 数据类型
## string
##### Redis的字符串是动态字符串，是可以修改的字符串
##### 内部结构实现上类似于 Java 的 ArrayList，采用预分配冗余空间的方式来减少内存的频繁分配
##### 内部为当前字符串实际分配的空间 capacity 一般要高于实际字符串长度 len。
##### 当字符串长度小于 1M 时，扩容都是加倍现有的空间，如果超过 1M，扩容时一次只会多扩 1M 的空间。
##### 需要注意的是字符串最大长度为 512M。
## list
